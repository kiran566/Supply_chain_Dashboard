
#  Supply Chain Inventory Dashboard (Power BI)

## Project Overview

This project presents an interactive **Supply Chain Inventory Dashboard** built using **Power BI** to monitor inventory performance, supplier efficiency, stock levels, and overall supply chain operations.

The objective of this dashboard is to provide actionable insights for better inventory planning, demand forecasting, and operational optimization.

---

## 🛠️ Tools & Technologies Used

*  Power BI
* Excel / CSV Dataset
*  DAX (Data Analysis Expressions)
*  Data Cleaning & Transformation (Power Query)

---

##  Dataset Description

The dataset contains supply chain and inventory-related information including:

* Product ID
* Product Name
* Category
* Supplier Name
* Warehouse Location
* Order Date
* Delivery Date
* Stock Quantity
* Reorder Level
* Lead Time
* Order Quantity
* Inventory Cost
* Sales Quantity

---

##  Project Objectives

* Monitor stock levels across warehouses
* Identify low-stock and overstock products
* Analyze supplier delivery performance
* Track inventory turnover
* Evaluate lead time impact on stock availability
* Improve decision-making with real-time KPIs

---

##  Dashboard KPIs

The dashboard includes the following key performance indicators:

* Total Inventory Value
*  Total Orders
* Inventory Turnover Ratio
* Average Lead Time
*  On-Time Delivery Rate
*  Stock-Out Frequency
*  Reorder Alerts

---

##  Dashboard Features

###  Inventory Overview

* Current stock levels by category
* Warehouse-wise stock distribution
* Low-stock alerts

### Supplier Performance Analysis

* Average delivery time
* On-time vs delayed deliveries
* Supplier comparison

###  Sales & Demand Trends

* Monthly demand trend
* Fast-moving vs slow-moving products
* Seasonal demand patterns

###  Cost Analysis

* Inventory holding cost
* Order cost trends
* Revenue vs inventory value

---

##  Key Insights

* Certain products frequently reach reorder levels, indicating high demand.
* Some suppliers show longer average lead times affecting stock availability.
* A small percentage of products contribute to majority of sales (Pareto principle).
* Overstocking observed in specific categories leading to higher holding costs.
